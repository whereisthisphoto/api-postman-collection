# API Postman Collection
This repo provides a Postman collection for the www.whereisthisphoto.com API.

How to use:
1. Download the json collection.
2. Import the collection into Postman.
3. Go to the collections variables.
4. Insert the API ID and secret key from http://whereisthisphoto.com/account?tab=api-keys into the variables.
5. You can now use the endpoints inside the collection to interact with the Where Is This Photo API.
